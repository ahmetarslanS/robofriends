# RoboFriends

Simple React project for learning purposes. It showcases a list of robot friends with search functionality.

* Displays a list of robot cards with names and emails. 
* Real-time search to filter robots by name.

---
* React
* React Hooks
* RoboHash API
* Tachyons
* Create React App
* Vite
---
## How To Run
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
